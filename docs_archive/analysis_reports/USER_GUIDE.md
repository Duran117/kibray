## Project Management

### Creating a Project

1. Go to **Projects** ‚Üí **New Project**
2. Fill in details:
   - **Project Name**: e.g., "Downtown Office Renovation"
   - **Client**: Select or create new
   - **Start Date** / **End Date**
   - **Budget**: Labor + Materials
   - **Description**: Project scope
3. Click **Create Project**

### Managing Tasks

#### Create Task

1. Open project
2. Click **Tasks** ‚Üí **Add Task**
3. Enter:
   - **Title**: Task name
   - **Description**: Details
   - **Assigned To**: Team member
   - **Due Date**: Deadline
   - **Priority**: Low/Medium/High/Critical
   - **Status**: Pending/In Progress/Completed
4. Save

#### Update Task Status

**Drag & Drop (Kanban)**:
- Drag task card to different column
- Columns: Pending ‚Üí In Progress ‚Üí Completed

**Quick Update**:
- Click task ‚Üí Change status dropdown
- Or use **Bulk Update** for multiple tasks

#### Task Attachments

- Click task ‚Üí **Attachments** tab
- Drag & drop photos
- Or click **Upload** button
- Supports: JPG, PNG, PDF (max 10MB)

### Gantt Chart

View project timeline:

1. Open project ‚Üí **Schedule** tab
2. Switch to **Gantt Chart** view
3. Features:
   - Drag bars to adjust dates
   - Click task to edit
   - Dependencies shown with arrows
   - Critical path highlighted in red

## Financial Management

### Creating Invoices

1. Go to **Finance** ‚Üí **Invoices** ‚Üí **New Invoice**
2. Select project
3. Add line items:
   - Labor hours (auto-filled from time entries)
   - Materials
   - Equipment rental
4. Review totals:
   - Subtotal
   - Tax (auto-calculated)
   - Total
5. Click **Generate Invoice**
6. **Send to Client**:
   - Email directly from Kibray
   - Or download PDF

### Tracking Expenses

1. **Finance** ‚Üí **Expenses** ‚Üí **Add Expense**
2. Enter:
   - **Date**: When incurred
   - **Project**: Related project
   - **Category**: Materials/Labor/Equipment
   - **Amount**: Cost
   - **Receipt**: Upload photo
3. Save

### Budget vs Actual

View financial health:

1. Open project ‚Üí **Financial** tab
2. See:
   - **Budget**: Original estimate
   - **Actual**: Spent to date
   - **Variance**: Over/Under budget
   - **% Complete**: Budget utilization
3. Charts:
   - Budget breakdown (pie chart)
   - Spending over time (line chart)
   - Earned Value Analysis

## Notifications

### Push Notifications

#### Enable on Desktop

1. First visit: Click **"Allow"** when prompted
2. Or go to **Settings** ‚Üí **Notifications**
3. Click **Enable Push Notifications**
4. Browser will ask for permission ‚Üí Allow

#### Enable on Mobile

**Android**:
- Automatically prompted after install
- Or: **Settings** ‚Üí **Notifications** ‚Üí Toggle on

**iOS** (if installed as PWA):
- **Settings** ‚Üí **Notifications**
- Find **Kibray**
- Toggle **Allow Notifications**

### Notification Types

- **Task Assigned**: When someone assigns you a task
- **Task Due Soon**: 24 hours before deadline
- **Task Overdue**: Daily reminder for overdue tasks
- **Message Received**: New chat message
- **@Mention**: Someone mentioned you
- **Status Change**: Task/project status changed
- **Payment Received**: Invoice paid
- **Budget Alert**: Project over budget

### Customizing Notifications

1. **Profile** ‚Üí **Settings** ‚Üí **Notifications**
2. Toggle each type on/off
3. Set preferences:
   - **Instant**: Real-time push
   - **Daily Digest**: Once per day summary
   - **Off**: No notifications

## Troubleshooting

### App Won't Install

**Android/Desktop**:
- Ensure you're on HTTPS
- Clear browser cache
- Try Chrome/Edge

**iOS**:
- Must use Safari (not Chrome)
- iOS 11.3+ required

### Notifications Not Working

1. Check browser/OS permissions:
   - **Chrome**: Settings ‚Üí Site Settings ‚Üí Notifications ‚Üí Allow
   - **iOS**: Settings ‚Üí Notifications ‚Üí Safari/Kibray ‚Üí Allow
2. Check Kibray settings: **Profile** ‚Üí **Notifications**
3. Try disabling and re-enabling

### Offline Mode Not Working

1. Close and reopen app
2. Check service worker:
   - Chrome DevTools ‚Üí Application ‚Üí Service Workers
   - Should show "activated and running"
3. Clear cache and reload

### Sync Conflicts

If changes conflict:
1. You'll see: "‚ö†Ô∏è Sync Conflict"
2. Click **Resolve**
3. Choose:
   - **Use Server Version**: Discard local changes
   - **Use My Version**: Overwrite server
   - **Merge**: Combine both (if possible)

### Performance Issues

- **Clear cache**: Settings ‚Üí Privacy ‚Üí Clear Data
- **Reduce animations**: Settings ‚Üí Accessibility ‚Üí Reduce Motion
- **Update browser**: Ensure latest version
- **Check internet speed**: Minimum 1 Mbps recommended
