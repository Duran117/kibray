Integración con Calendario
Descargar ICS

Descarga un archivo .ics que puedes importar en cualquier aplicación de calendario:

  • Google Calendar: Abre Google Calendar → Configuración → Importar y exportar → Importar
  • Outlook: Archivo → Abrir y exportar → Importar/Exportar → Importar archivo iCalendar
  • Apple Calendar: Archivo → Importar
Google Calendar (Suscripción)

Suscríbete al cronograma para recibir actualizaciones automáticas cuando cambien las fechas.

Los eventos se sincronizan con los colores según el estado: verde (completado), azul (en progreso), rojo (bloqueado).

Los hitos aparecen como eventos de día completo. Las tareas regulares muestran el rango de fechas planificado.
