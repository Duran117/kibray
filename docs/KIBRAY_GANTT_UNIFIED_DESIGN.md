## ğŸ”§ Technical Decisions

### Why React + TypeScript?
- Already have React infrastructure (Frappe Gantt, NotificationCenter)
- Type safety for complex state management
- Better IDE support for large components

### Why Vite?
- Fast HMR during development
- Easy to integrate with Django (static file output)
- Modern ES modules support

### Why Custom vs Library?
- Full control over UX (Monday.com style)
- Integration with existing API/models
- No vendor lock-in
- Libraries like DHTMLX are paid for full features

### Drag & Drop Approach
- Using native HTML5 drag events + CSS transforms
- `react-dnd` as potential fallback if needed
- Debounced API updates (300ms after drag ends)
