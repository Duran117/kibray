# Política de seguridad

Si encuentras una vulnerabilidad de seguridad, por favor repórtala a: seguridad@ejemplo.com o utiliza el sistema de GitHub Security Advisories.

Por favor incluye:
- Descripción de la vulnerabilidad
- Pasos para reproducir
- Severidad estimada
- Posible parche o mitigación

Responderemos en un plazo razonable y coordinaremos divulgación responsable.
